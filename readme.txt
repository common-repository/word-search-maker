== Description ==

Create a word search puzzle in a post or in a page, using your own list of words, and your own dimensions.

== Installation ==

1. Upload the 3 files in the zip to `/wp-content/plugins/wordsearchmaker/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use example tag found in Word Search Maker's description in a WordPress page or post.
